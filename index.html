oriscript encoder

Oriscript is a simple conscript which can be written with almost anything, even by folding paper.
To learn how to read it, please check out the original project page

This Oriscript encoder only encodes the 26 letters of the English alphabet and "alpha" to denote a "space"

made for Oriscript's submission to omniglot.com

Oriscript is an invention from Christopher Shelley
